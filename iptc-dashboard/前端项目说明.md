# AI赋能思政案例库 - 前端项目

基于Vue 3 + TypeScript + Vite构建的思政案例库与知识图谱可视化系统。

## 技术栈

- **框架**: Vue 3.4 (Composition API + `<script setup>`)
- **语言**: TypeScript 5.3
- **构建工具**: Vite 5.0
- **路由**: Vue Router 4.3
- **状态管理**: Pinia 2.1
- **HTTP客户端**: Axios 1.6
- **图谱可视化**: @antv/g6 4.8
- **样式**: CSS3 + CSS Variables

## 核心功能

### 1. 首页 (HomeView)
- 英雄区域（Hero Section）
- 核心功能介绍
- 平台数据统计
- 动画效果

### 2. 案例库 (CasesView)
- 搜索功能（防抖搜索）
- 筛选器（知识点、时间范围、排序）
- 卡片网格布局
- 分页导航
- 空状态处理

### 3. 案例详情 (CaseDetailView)
- 面包屑导航
- 案例内容展示
- 知识点标签
- 相关案例推荐
- 分享和打印功能

### 4. 知识图谱 (GraphView)
- G6图谱可视化
- 力导向布局（支持多种布局切换）
- 节点交互（点击、拖拽、缩放）
- 节点筛选（案例/知识点）
- 控制面板
- 图谱统计
- 导出图片

### 5. 关于页面 (AboutView)
- 项目介绍
- 核心功能说明
- 联系方式

## 设计规范

### 色彩系统
- 主色：深蓝色 `#1a365d` - 代表权威、知识
- 辅助色：中国红 `#d03050` - 体现思政特色
- 点缀色：青绿色 `#36cfc9` - 科技感、活力

### 间距系统
基于8px基准：4px, 8px, 12px, 16px, 24px, 32px, 48px, 64px

### 圆角系统
- 小圆角：4px
- 中等圆角：8px
- 卡片圆角：12px

### 阴影系统
- 轻阴影：`0 2px 8px rgba(0,0,0,0.08)`
- 中阴影：`0 4px 12px rgba(0,0,0,0.12)`
- 重阴影：`0 8px 24px rgba(0,0,0,0.16)`

### 响应式断点
- xs: < 576px (手机)
- sm: ≥ 576px (大手机)
- md: ≥ 768px (平板)
- lg: ≥ 992px (小桌面)
- xl: ≥ 1200px (桌面)
- xxl: ≥ 1600px (大桌面)

## API配置

API基础URL配置在 `src/api/request.ts` 中：
```typescript
baseURL: 'http://localhost:11528'
```

## 状态管理

使用Pinia进行状态管理，主要Store包括：

- **useAppStore**: 全局应用状态、统计数据
- **useCaseStore**: 案例数据、搜索状态、筛选条件
- **useGraphStore**: 图谱数据、节点选择、布局配置
- **useUserStore**: 用户信息、登录状态

## 组合式函数

- **useSearch**: 搜索功能（防抖、最小长度验证）
- **useGraph**: 图谱可视化（G6集成、布局管理、节点交互）
- **usePagination**: 分页逻辑（页码管理、总页数计算）

## 工具函数

- **date.ts**: 日期格式化、相对时间
- **common.ts**: 防抖、节流、深度克隆、文件下载、剪贴板操作

## 动画效果

- 页面过渡：淡入淡出 + 滑动
- 卡片悬停：上浮 + 阴影加深
- 骨架屏加载动画
- 空状态展示

## 注意事项

1. 所有API请求都经过统一拦截器处理
2. 图谱可视化使用@antv/g6，需要注意浏览器兼容性
3. 响应式设计支持移动端和桌面端
4. 使用TypeScript确保类型安全
5. 遵循Vue 3 Composition API最佳实践
