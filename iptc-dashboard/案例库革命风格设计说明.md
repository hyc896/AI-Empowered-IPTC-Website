# IPTC案例库革命风格重新设计说明

## 🎨 设计理念

本次重新设计以"传承红色基因、铸就时代新人"为核心理念，将革命先辈的奋斗精神融入界面设计，打造大气简约、富有动态效果的思政案例展示平台。

## ✨ 核心功能

### 1. 自动刷新机制
- **刷新频率**: 每小时自动刷新一次案例数据
- **进度指示**: 右下角刷新指示器实时显示刷新进度
- **智能更新**: 保持页面状态，无需用户手动操作

### 2. 革命主题视觉设计

#### 背景设计
- 使用革命题材背景图（五星红旗、革命先辈剪影、和平鸽）
- 红色渐变遮罩层（#D03050 → #8B0000 → #D03050）
- 毛玻璃效果（backdrop-filter）增强内容层次

#### 配色方案
- **主色**: 革命红 (#D03050, #8B0000)
- **金色**: 五星黄 (#FFD700) - 用于强调和高亮
- **白色**: 纯净白 (#FFFFFF) - 文字和卡片背景
- **渐变**: 红金渐变 - 体现革命精神与光明未来

### 3. 丰富的动态效果

#### 页面加载动画
- **英雄区域**: 渐入动画 (heroFadeIn)
- **搜索区域**: 上滑动画 (slideUp)
- **案例卡片**: 逐个渐入，带延迟效果

#### 交互动效
- **搜索按钮**: 光泽扫过效果 (shine)
- **金色标题**: 渐变流动动画 (goldShine)
- **旗帜加载**: 飘扬摆动效果 (waveFlag)
- **刷新图标**: 持续旋转动画 (rotate)

#### 悬停效果
- **案例卡片**: 上浮 + 放大 + 金色光晕
- **红色装饰条**: 宽度扩展至全屏
- **标题颜色**: 白色 → 金色过渡
- **箭头图标**: 向右移动效果

### 4. 英雄区域统计

展示三项核心数据：
- **案例总数**: 实时显示数据库案例数量
- **知识点**: 61个思政理论知识点
- **最近更新**: 显示最后更新时间（时:分）

### 5. 搜索与筛选

## 🔄 自动刷新机制详解

### 实现原理
```typescript
// 每小时刷新一次 (3600000ms)
const startAutoRefresh = () => {
  refreshInterval = window.setInterval(() => {
    caseStore.fetchCases();        // 重新获取案例数据
    updateLastUpdateTime();         // 更新时间显示
    refreshProgress.value = 0;      // 重置进度条
  }, 3600000);
};

// 进度条每秒更新一次
const startProgressAnimation = () => {
  progressInterval = window.setInterval(() => {
    refreshProgress.value = (refreshProgress.value + 0.028) % 100;
  }, 1000);
};
```

### 刷新指示器
- 位置: 固定在右下角
- 图标: 🔄 持续旋转
- 文字: "每小时自动更新"
- 进度条: 红金渐变，100秒循环
