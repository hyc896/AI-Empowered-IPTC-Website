# 思政课智能案例系统前端

## 技术栈

- Vue 3.4+ (Composition API)
- TypeScript 5.0+
- Vite 5.0+
- Element Plus 2.5+
- AntV G6 4.8+
- Pinia 2.1+
- Vue Router 4.3+

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览构建产物
npm run preview
```

## 访问地址

开发环境: http://localhost:5174

## 目录结构

```
src/
├── views/          # 页面视图
├── components/     # 组件
├── api/           # API接口
├── stores/        # 状态管理
├── router/        # 路由配置
├── types/         # TypeScript类型
├── utils/         # 工具函数
├── composables/   # Vue组合式函数
└── assets/        # 静态资源
```

## 主要功能

1. 案例库浏览和检索
2. 知识图谱可视化展示
