# 搜索爬虫使用指南

## 快速开始

搜索并爬取指定关键词的书籍：

```bash
python scripts/search_libsp.py
```

然后输入搜索关键词，如：人工智能、法学、经济学等

## 命令行直接搜索

```bash
echo "人工智能" | python scripts/search_libsp.py
```

## 功能特点

- 支持任意关键词搜索
- 自动滚动加载更多结果
- 提取书名、封面图片URL
- 导出JSON和CSV两种格式

## 输出示例

搜索"人工智能"后会生成：

- `search_results_YYYYMMDD_HHMMSS.json` - JSON格式数据
- `search_results_YYYYMMDD_HHMMSS.csv` - CSV格式数据

## 数据字段

- `title` - 书名
- `author` - 作者（如有）
- `cover` - 封面图片URL
- `link` - 书籍链接（如有）

## 示例结果

```json
[
  {
    "title": "人工智能: 技术与经济的双重奇点",
    "cover": "https://cover01.chaoxing.com/...",
    "link": null
  }

```

## 其他脚本

- `crawl_libsp.py` - 爬取首页书籍
- `analyze_libsp.py` - 分析网站结构
- `test_api.py` - 测试API接口
