# 类型定义目录说明

## 目录作用

types 目录包含 TypeScript 类型定义文件，为整个应用提供类型安全保障。

## 文件列表

### 核心文件

- `cosma.ts` - Cosma 图谱相关类型定义
  - 节点（Node）类型定义
  - 边（Link）类型定义
  - 图谱配置类型
  - 搜索结果类型

- `index.ts` - 通用类型定义
  - API 响应类型
  - 文件上传类型
  - 配置类型
  - 其他通用类型

## 技术说明

### 类型定义原则

- 使用 interface 定义对象结构
- 使用 type 定义联合类型和工具类型
- 导出所有公共类型
- 为复杂类型添加注释说明

### Cosma 类型结构

```typescript
interface CosmaNode {
  id: string
  label: string
  type: string
  // ...其他属性
}

interface CosmaLink {
  source: string
  target: string
  type: string
  // ...其他属性
}
```

## 使用示例

### 定义新类型

```typescript
// 在 index.ts 中定义
export interface UploadResponse {
  file_id: string
  filename: string
  status: string
}
```

### 使用类型

```typescript
import type { UploadResponse } from '@/types'

const response: UploadResponse = await uploadFile(file)
```

## 扩展方式

### 添加新的类型定义

1. 在对应的类型文件中添加新的 interface 或 type
2. 导出类型供其他模块使用
3. 为复杂类型添加 JSDoc 注释

## 注意事项

### 类型定义规范

- 使用描述性的类型名称
- 为可选属性使用 `?` 标记
- 避免使用 `any` 类型
- 复杂类型拆分为多个小类型
- 为类型添加注释说明
