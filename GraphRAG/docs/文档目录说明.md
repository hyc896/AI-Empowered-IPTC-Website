# 文档目录说明

## 目录作用

docs目录包含GraphRAG工具包的详细文档，提供完整的使用指南、API参考和技术说明。

## 文件列表

- `工具包说明.md` - 总体说明文档，包含快速开始、核心概念、安装指南
- `核心模块说明.md` - 核心模块详细文档，与graphrag/core/核心模块目录说明.md内容一致
- `工具模块说明.md` - 工具模块详细文档，与graphrag/utils/工具模块目录说明.md内容一致
- `LLM集成说明.md` - LLM集成指南，与graphrag/llm/LLM集成目录说明.md内容一致
- `API参考.md` - API接口文档，包含所有类和方法的详细说明

## 技术说明

### 文档结构

文档采用分层结构：
1. **总体说明**：快速开始和核心概念
2. **模块说明**：各模块的详细文档
3. **API参考**：完整的API文档

### 文档格式

使用Markdown格式：
- 清晰的标题层级
- 代码示例
- 表格和列表
- 链接和引用

## 使用示例

### 阅读文档

按照以下顺序阅读：
1. 工具包说明.md - 了解整体架构
2. 核心模块说明.md - 学习核心功能
3. LLM集成说明.md - 配置LLM
4. API参考.md - 查阅具体API
