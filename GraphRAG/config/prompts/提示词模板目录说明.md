# 提示词模板目录说明

## 目录作用

prompts目录包含GraphRAG使用的LLM提示词模板，用于指导LLM进行实体提取、关系识别等任务。

## 文件列表

- `entity_extraction.txt` - 实体提取提示词模板

## 技术说明

### 提示词设计原则

1. **清晰的角色定义**：明确LLM的角色和任务
2. **详细的规则说明**：提供具体的提取规则
3. **丰富的示例**：展示期望的输出格式
4. **格式约束**：要求严格的JSON格式输出

### 实体提取提示词结构

**包含内容**：
- 角色定义
- 实体类型说明（8种）
- 关系类型说明（7种）
- 提取规则（6条）
- 输出格式示例

**实体类型**：
- Company（公司）
- Person（人物）
- Technology（技术）
- Product（产品）
- Organization（组织）
- Event（事件）
- Concept（概念）
- Location（地点）

**关系类型**：
- WORKS_AT（任职）
- INVESTS_IN（投资）
- DEVELOPS（开发）
- COMPETES_WITH（竞争）
- PARTNERS_WITH（合作）
- BASED_ON（基于）
- LOCATED_IN（位于）

## 使用示例

### 在代码中使用提示词

```python
from pathlib import Path

# 读取提示词模板
prompt_path = Path("config/prompts/entity_extraction.txt")
with open(prompt_path, 'r', encoding='utf-8') as f:
    system_prompt = f.read()

# 构建消息
messages = [
    {"role": "system", "content": system_prompt},
    {"role": "user", "content": f"请从以下文本中提取实体和关系：\n\n{text}"}
]

# 调用LLM
response = await llm_client.generate_async(messages)
```

### 自定义提示词

创建新的提示词文件：

```
# custom_extraction.txt
你是一个专业的领域知识图谱构建助手。

## 领域特定实体类型
1. YourType1（说明）
2. YourType2（说明）

## 领域特定关系类型
- YOUR_RELATION1（说明）
- YOUR_RELATION2（说明）

## 提取规则
1. 你的规则1
2. 你的规则2
```

## 扩展方式

### 添加新的实体类型

在`entity_extraction.txt`中添加：

```
## 实体类型
...
9. YourNewType（新类型说明）
```

### 添加新的关系类型

在`entity_extraction.txt`中添加：

```
## 关系类型
...
- YOUR_NEW_RELATION（新关系说明）
```

### 创建领域特定提示词

针对特定领域创建专门的提示词：

```
# medical_extraction.txt
你是一个医学知识图谱构建助手。

## 医学实体类型
1. Disease（疾病）
2. Drug（药物）
3. Symptom（症状）
4. Treatment（治疗方法）

## 医学关系类型
- TREATS（治疗）
- CAUSES（导致）
- PREVENTS（预防）
```

## 注意事项

### 提示词优化

- 根据实际效果迭代优化
- 提供清晰的示例
- 避免过于复杂的指令
- 测试不同模型的表现

### 幻觉控制

- 强调"仅提取明确出现的实体"
- 禁止推断潜在关系
- 要求严格的JSON格式
- 避免引导式语言

### 多语言支持

- 提供中英文双语提示词
- 根据输入语言选择提示词
- 确保输出格式一致

### 版本管理

- 记录提示词变更历史
- 测试新版本的效果
- 保留旧版本作为备份
